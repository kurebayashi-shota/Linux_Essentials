# コマンド連続実行まとめ

| カテゴリ         | 記号    | 主な用途                   | 説明                                                                  |
| ---------------- | ------- | -------------------------- | --------------------------------------------------------------------- |
| コマンド連続実行 | `;`     | 単純な連続実行             | 直前のコマンドの成功・失敗に関係なく次を実行                          |
| コマンド連続実行 | `&&`    | 条件付き連続実行（成功時） | 直前のコマンドが**成功（終了ステータス 0）した場合のみ**次を実行      |
| コマンド連続実行 | `\|\|`※ | 条件付き連続実行（失敗時） | 直前のコマンドが**失敗（終了ステータス 0 以外）した場合のみ**次を実行 |

- MAIL=/var/spool/mail/ormaster
  <!-- ユーザーのメールスプール -->

- HISTFILE=/home/ormaster/.bash_history
  <!-- コマンド履歴を保存するファイル -->

- HISTSIZE=1000
  <!-- シェルが保持するコマンド履歴数 -->
- HISTFILESIZE
<!-- コマンド実行履歴保存ファイルの最大記録数を設定する環境変数 -->

3. 違いのイメージ
   変数名 保持場所 内容
   HISTSIZE メモリ上 現在のシェルセッションの履歴件数
   HISTFILESIZE ファイル (~/.bash_history) 永続化される履歴の最大件数

HISTSIZE < HISTFILESIZE の場合、メモリ上は少ない件数だけ保持されるが、履歴ファイルにはより多く保存可能

HISTSIZE > HISTFILESIZE の場合、履歴ファイルの方が少ない件数しか保存されない

# tar コマンドの基本

1️⃣ tar の基本アクション
文字 意味 例
-c create（作成） tar -czvf sample.tar.gz dir/
-x extract（展開） tar -xzvf sample.tar.gz
-t table/list（中身確認） tar -tzvf sample.tar.gz

ポイント：この 3 つは必ずどれか 1 つを指定する。これが軸

2️⃣ 圧縮方式
文字 意味 例
-z gzip 圧縮 (.tar.gz) tar -czvf / tar -xzvf
-j bzip2 圧縮 (.tar.bz2) tar -cjvf / tar -xjvf
-J xz 圧縮 (.tar.xz) tar -cJvf / tar -xJvf

「c/x/t」 + 「圧縮オプション」 + 「v」 + 「f」 でほぼ完結。

3️⃣ よく使うオプション（補助）
文字/オプション 意味 使い方例
-v verbose（詳細表示） tar -cvzf sample.tar.gz dir/
-f ファイル名指定 tar -czvf sample.tar.gz dir/
-C 展開先ディレクトリ指定 tar -xzvf sample.tar.gz -C out/
--exclude 除外パターン tar -czvf sample.tar.gz dir/ --exclude='\*.log'
